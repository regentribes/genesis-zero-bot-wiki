# Agent Skills

**Status:** active
**Date:** 2026-05-07
**Source:** [View source](/genesis-zero-bot-wiki/sources/0012-github-trends-may-2026.html)

## Key Claims

- Agent skills are reusable tool/action modules that AI agents use to extend their capabilities

## Summary

Agent skills are the emerging software artifact category for AI agent extensibility. They are packaged capabilities (tools, actions, prompts) that agents can load and use.

Key platforms:
- agentskills.io — centralized registry for agent skills
- awesome-agent-skills — community curated list with 1,000+ entries
- Anthropic official guide for skill development

Supported by major agent platforms: Claude Code, Cursor, Gemini CLI, Codex.
