# ADR 0007 — Kabanov Continuous Attention AGI Model

**Status:** proposed  
**Date:** 2026-05-06  
**Full document:** [[adrs/0007-kabanov-continuous-attention-agi|View ADR]]  
**Source:** [[sources/0007-kabanov-continuous-attention-agi|Source page]]

## Key Decision

Kabanov continuous attention model adopted as theoretical substrate for AME processing, ITC reasoning, and agent memory design — over static transformer attention.

## Core Insight

- **Atom of cognition:** Transition, not object. Trajectories through phase space define perception.
- **Three outcomes:** Confirmation, refutation (new branch), uncertainty (timeout)
- **Gierarchical competition:** Multiple super-generalizations compete for attention — not a tree, a marketplace
- **System broker:** Ranks competing requests, builds optimal navigation routes
- **Intermittency:** Multiple trajectories coexist while each stays above noise floor

## Relevance to RegenTribes

- AME seed activation ← maps to peripheral driver + system broker bootstrap
- ITC reasoning ← maps to gierarchical competition + three-outcome loop
- FOT trust scoring ← maps to trajectory confirmation frequency
- Agent memory ← maps to dream-state consolidation (14,000+ generalization binary tree)
