## Key Claims

- AI agents are the dominant paradigm shift in GitHub trending repositories as of May 2026

## Summary

AI agents are LLM-powered systems that autonomously use tools, execute multi-step tasks, and interact with external environments (files, browsers, APIs, terminals). The paradigm shifted from chat assistants to agentic systems in 2025-2026.

Key enablers:
- Tool use protocols (MCP, Anthropic tool use, OpenAI function calling)
- Agent frameworks (Claude Code, Cursor, Gemini CLI, Codex)
- Agent skills ecosystem (agentskills.io, awesome-agent-skills with 1,000+ entries)
