**Status:** active
**Date:** 2026-05-07

## Key Claims

- MCP is the critical infrastructure protocol for AI agent tool integration

## Summary

MCP (Model Context Protocol) is a standardized protocol enabling AI models to connect with external tools, data sources, and services. It provides a universal interface for agent tool use.

Key properties:
- Standardized tool discovery and invocation
- Context enrichment from external sources
- Safe permission model for tool access
